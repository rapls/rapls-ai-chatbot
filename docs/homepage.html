よくある質問

Free 版だけで使えますか？

はい。Free 版だけで4つの AI プロバイダー、サイト学習、ベクトル RAG、Web 検索、ナレッジベース、6テーマ、Gutenberg ブロック、MCP Server など主要機能がすべて制限なしで使えます。Pro Add-on はアナリティクス、リードキャプチャ、WooCommerce 連携、LINE 連携などの高度なビジネス機能を追加します。

Pro はサブスクリプションですか？

いいえ、買い切りです。1年間のアップデートとサポートが含まれます。2年目以降は年 $19 で延長可能ですが、延長しなくても引き続き使えます。

API キーはどこで取得できますか？

各 AI プロバイダーの公式サイトから取得できます。OpenAI は platform.openai.com、Claude は console.anthropic.com、Gemini は aistudio.google.com、OpenRouter は openrouter.ai にアクセスしてください。

どの AI モデルがおすすめですか？

コストと性能のバランスなら「GPT-4o-mini」や「Gemini 2.5 Flash」がおすすめです。最高性能を求めるなら「Claude Opus 4.6」や「GPT-5.2」が適しています。

MCP Server とは何ですか？

MCP（Model Context Protocol）Server は、Claude Desktop や Cursor などの AI ツールからあなたの WordPress サイトを直接操作できる機能です。投稿の作成・検索、ナレッジベースの管理など7つのツールを提供します。

Free 版から Pro にアップグレードするには？

Pro Add-on を追加インストールするだけです。Free 版の設定やデータはそのまま引き継がれます。今なら Early Bird 価格 $29（通常 $59）でご購入いただけます。
