# AIチャットボット 管理画面 全体構造

---

## Free版

### メニュー構成（9ページ）

#### 1. ダッシュボード (`wpaic-dashboard`)
- 統計カード: 会話数 / 今日のメッセージ / インデックス済みページ / ナレッジ数 / 月間AI応答数(制限)
- ステータス: AIプロバイダー / サイト学習 / 履歴保存
- API使用統計（過去30日）: トークン合計 / 入力トークン / 出力トークン / 推定コスト / チャート / プロバイダー別内訳

#### 2. 設定 (`wpaic-settings`) — 5タブ

**タブ1: AI設定**
- AIプロバイダー選択（OpenAI / Claude / Gemini / OpenRouter）
- プロバイダー別設定: APIキー / モデル選択 / 接続テスト
- ベクトル検索（RAG）: 有効化 / 埋め込みプロバイダー / ステータス
- MCP: 有効化 / APIキー生成 / エンドポイント / Claude Desktop設定例

**タブ2: チャット設定**
- ボット名 / アバター（絵文字 or 画像）
- ウェルカムメッセージ（言語別設定あり：英/日/中/韓/西/仏/独/葡/伊/露/アラビア/タイ/ベトナム）
- システムプロンプト / 応答言語 / メッセージ履歴数
- フィードバックボタン / APIクォータエラーメッセージ
- コンテキストプロンプト（詳細）: ナレッジ完全一致 / Q&A形式 / サイト学習コンテキスト
- 機能プロンプト（詳細）: 再生成指示 / 良い例からの学習 / 悪い例からの学習 / 会話要約

**タブ3: 表示設定**
- テーマ選択（Free 6種: Default / Simple / Classic / Light / Minimal / Flat、Pro 10種はロック表示）
- バッジ位置（4隅グリッド） / バッジ余白 / バッジアイコン（Proロック）
- プライマリカラー / セカンダリカラー
- モバイル表示 / Markdown / タイピングインジケーター / 最大入力長
- ページ除外 / フッター
- クロスサイト埋め込み: Script埋め込み / Iframe埋め込み

**タブ4: セキュリティ設定**
- reCAPTCHA: 有効化 / サイトキー / シークレットキー / スコア閾値 / 既存reCAPTCHA利用
- アクセス制御: 同意厳格モード / レート制限 / Cloudflare連携 / リバースプロキシ信頼 / reCAPTCHA失敗モード
- セキュリティ診断（読み取り専用）: 許可オリジン / 信頼プロキシ / IP検出 / APIキー状態 / WP Consent API / レート制限 / reCAPTCHA / SSL/TLS / CSRF

**タブ5: データ管理**
- 会話履歴: 保存有効化 / 保持日数 / アンインストール時データ削除
- インポート/エクスポート: 設定エクスポート（ナレッジ含む選択） / 設定インポート / 設定リセット

#### 3. ナレッジ (`wpaic-knowledge`)
- ナレッジ制限表示（Free: X/Y件）
- テキスト追加フォーム: タイトル / 内容 / カテゴリー / 優先度 / タイプ（Proロック）
- ファイルインポート: ファイルアップロード（.txt/.csv/.md/.pdf/.docx） / カテゴリー
- 統計カード: 合計 / 有効 / 無効 / カテゴリー数
- ナレッジ一覧: ステータスフィルタ（すべて / 公開 / 下書き） / テーブル（ID / タイトル / カテゴリー / タイプ / 優先度 / 更新日 / 操作）

#### 4. Pro設定 (`wpaic-pro-settings`) — アップセルページ（ロック表示）

#### 5. サイト学習 (`wpaic-crawler`) — アップセルページ（ロック表示）

#### 6. 会話履歴 (`wpaic-conversations`) — アップセルページ（ロック表示）

#### 7. 分析 (`wpaic-analytics`) — プレビュー付きアップセルページ
- サンプルデータのプレビュー表示（統計カード / チャート / テーブル等）

#### 8. リード (`wpaic-leads`) — プレビュー付きアップセルページ

#### 9. 監査ログ (`wpaic-audit-log`) — アップセルページ（ロック表示）

#### 5. サイト学習 (`wpaic-crawler`) — Pro版で実機能が有効

- よくある質問テーブル / トップページテーブル
