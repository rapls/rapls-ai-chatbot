Key Features

4 AI Providers

Web SearchNEW

When the knowledge base lacks information, AI automatically searches the web. Uses provider-native search with no additional API key required.

Vector RAGNEW

Hybrid search combining FULLTEXT (40%) and vector search (60%) for highly accurate contextual understanding and responses.

Gutenberg Block

Fully integrated with the block editor. SSR (server-side rendering) support for easy chatbot placement within posts and pages.

MCP ServerNEW

Built-in MCP Server with 7 tools. Operate WordPress directly from Claude Desktop, Cursor, or VS Code.

6 Free Themes

Default, Simple, Classic, Light, Minimal, and Flat themes included. Customize to match your site's design.

Replies in 13 Languages

English, Japanese, Chinese, Korean, Spanish, French, German and more — 13 languages auto-detected from the visitor's browser. Especially valuable for tourism, hospitality, and retail sites that can't staff every language at the front desk.

Preset Question ButtonsNEW

Up to 10 one-tap chips under the welcome message — "Pricing", "Hours", "Staff perks" — so visitors don't have to type. Canned reply mode bypasses the AI entirely for chosen chips: zero tokens, instant response, fully scripted wording. Pro adds hierarchical groups (parent ▸ children) for navigation-style bots.

Glossary (Proper Nouns)NEW

Register up to 50 product / service / brand names you don't want the AI translating or rewording — no more "Rapls Works" coming back as "Rapls Workshop" in English replies. Optional per-language overrides ("EN: Staff Perks, ZH: 员工福利") let you pin a specific translation per language.

Frequently Asked Questions

Can I use just the free version?

Yes. The free version includes 4 AI providers, site learning, Vector RAG, Web Search, Knowledge Base, 6 themes, Gutenberg block, MCP Server, and more — all with no artificial limits. Pro Add-on adds analytics, lead capture, WooCommerce integration, LINE, and other advanced business features.

Is Pro a subscription?

No, it is a one-time purchase. Includes 1 year of updates and support. You can renew at $19/year, but the plugin continues to work even without renewal.

Where can I get API keys?

From each AI provider's official site: OpenAI at platform.openai.com, Claude at console.anthropic.com, Gemini at aistudio.google.com, and OpenRouter at openrouter.ai.

Which AI model do you recommend?

For the best cost-performance balance, try GPT-4o-mini or Gemini 2.5 Flash. For top performance, Claude Opus 4.6 or GPT-5.2 are excellent choices.

What is MCP Server?

MCP (Model Context Protocol) Server lets AI tools like Claude Desktop, Cursor, and VS Code directly interact with your WordPress site. It provides 7 tools for creating/searching posts, managing knowledge base, and more.

How do I upgrade from Free to Pro?

Simply install the Pro Add-on plugin alongside the free version. All your settings and data are preserved. Available now at the Early Bird price of $29 (regular $59).
