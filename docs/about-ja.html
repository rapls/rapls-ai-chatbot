Rapls Works について

Rapls Works は、WordPress プラグインや Web ソリューションの開発を行っています。最新の AI 技術を活用し、使いやすく高品質な製品を提供することを目指しています。

製品

Rapls AI Chatbot

WordPress 用 AI チャットボットプラグイン。OpenAI（GPT-5 シリーズ）、Anthropic Claude（Opus 4.6/Sonnet 4.5）、Google Gemini（3/2.5 シリーズ）、OpenRouter（100以上のモデル）に対応。Web 検索、ベクトル RAG、Gutenberg ブロック、MCP サーバー、WordPress Abilities API ブリッジを搭載しています。

Free マニュアル Pro マニュアル WordPress.org

Rapls PDF Image Creator

WordPress 用 PDF 画像変換プラグイン。ブロックエディタ内で PDF ページを高品質な画像に変換できます。
